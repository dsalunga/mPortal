# Canonical Cards

This folder is split into two tracks:

- [Legacy canonical cards](./legacy/README.md): source-of-truth analysis of the current `legacy/` implementation.
- [Modern canonical cards](./modern/README.md): target-state architecture and migration direction for the `.NET 10` platform.

Recommended workflow:
1. Read the `legacy` card for current behavior and constraints.
2. Read the matching `modern` card for target architecture and implementation direction.
3. Use both together during migration planning, execution, and validation.
